## Critical Gaps

### GAP-001: Navigation Routing Mismatch

**Severity:** CRITICAL
**Impact:** Users cannot access Bug Dashboard via sidebar navigation

**Details:**
- `src/pages/Layout.jsx:164` uses `createPageUrl("BugDashboard")` which produces `/bugdashboard`
- `src/pages/index.jsx:194` defines route as `path="/bugs"`
- These don't match, so sidebar link goes nowhere

**Fix Options:**

**Option A (Recommended):** Change route to match convention
```jsx
// src/pages/index.jsx line 194
// Change from:
<Route path="/bugs" element={<BugDashboard />} />
// To:
<Route path="/bugdashboard" element={<BugDashboard />} />
```

**Option B:** Hardcode href in Layout.jsx
```jsx
// src/pages/Layout.jsx line 162-166
// Change from:
href: createPageUrl("BugDashboard"),
// To:
href: "/bugs",
```

Option A is preferred as it follows the existing convention where `createPageUrl(PageName)` produces the route path.
